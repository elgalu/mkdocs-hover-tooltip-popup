# D2 Diagrams

This section contains examples and documentation for D2 diagram integration with the Hover Tooltip Popup plugin.

## About D2

D2 is a modern diagram scripting language that turns text to diagrams. The Hover Tooltip Popup plugin provides seamless
zoom and pan functionality for D2 diagrams.

## Features

- **Automatic Detection**: D2 diagrams are automatically detected and enhanced with panzoom
- **Zoom & Pan**: Navigate large and complex D2 diagrams with ease
- **File Import**: Support for imported D2 diagram files
