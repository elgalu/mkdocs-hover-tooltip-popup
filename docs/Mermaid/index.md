# Mermaid Diagrams

This section contains examples and documentation for Mermaid diagram integration with the Hover Tooltip Popup plugin.

## Features

- **Smart Auto-Detection**: Automatically enables/disables panzoom based on diagram complexity
- **Manual Override**: Control panzoom behavior with YAML metadata
- **Large Diagram Support**: Enhanced viewing for complex diagrams
- **Interactive Elements**: Support for clickable elements and links
- **Exclusion Options**: Ability to exclude specific diagrams from panzoom
