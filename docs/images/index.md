# Images

This section contains examples and documentation for image integration with the Hover Tooltip Popup plugin.

## Features

The Hover Tooltip Popup plugin can enhance images with zoom and pan functionality:

- **Large Image Support**: Navigate high-resolution images with ease
- **Zoom Controls**: Zoom in and out of image details
- **Pan Navigation**: Pan around large images
- **Keyboard & Mouse Support**: Full interaction support

## Configuration

To enable panzoom for images, add `img` to your include selectors:

```yaml
plugins:
  - hover-tooltip-popup:
      include_selectors:
        - "img"
```
